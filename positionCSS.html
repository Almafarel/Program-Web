position static

Position static digunakan untuk mengatur elemnt menjadi statis secara default. Elemnt akan mengikuti posisi normal secara default, elemnt tidak dipenagruhi oleh propteri seperti top, bottom, left, & right.:

Elemnt ini diatur menggunakan posisi static;




position: relative;

Sebuah elemnt HTML yg menggunakan position relative akan terletak pada posisi normal. Meengatur propteri atas, kanan, bawah, dan kiri dari elemnt yg diposisikkan dengan "relatif" akan membautanya jauh dari posisi normalnaya. Konten lain tidak akan disesuaikan agar sesuai dengan celah yg ditinggalkan oleh elemnt tsb.:

Elemnt ini diatur menggunakan posisi : relative;



position: fixed;

Sebuah HTML yg di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman web di scroll. Berlaku pengaturan left, bottom, top, dan right pada position fixed:

Elemnt ini diatur menggunakan posisi : fixed;















position: absolute;

Elemnt HTML yg menggunakan position absolute akan diposisikkan relatif dengan elemnt lain yg mndhluinya yg terdekat, bukan relative terhadap layar secara normal.

Elemnt ini diatur menggunakan posisi: relative:
Elemnt ini diatur menggunakan posisi: absolute;






position: sticky;

Cobalah untuk scroll untuk mengethui bagaimna posisi sticky bekerja.

Catatan: IE/Edge 15 dan vrsi yg lebih lama tidak mendukung sticky.

Hola aku adlah sticky!

Dalam contoh ini, elemen sticky menempel pada posisi paling atas halaman (top:0), ketika mencapai posisi scrollnya.
